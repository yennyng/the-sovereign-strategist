Chapter 1 · The Quiet Choice

I once thought power was the loud kind — a voice that took the center and did not ask permission. But the most sustainable decisions have been the quiet ones, the ones I rehearsed in the margins of my notebook.
⋯⋯⋯⋯⋯⋯⋯⋯⋯⋯
我曾以为权力是那种喧闹的存在——占据中心、不需征求许可的声音。但真正可持续的决定，经常是那些静默的选择，是我在笔记边缘反复排练的小心思。
This chapter keeps small rituals: an hour of reading, a slow reply, a deliberate refusal. Each one is a strategy disguised as a habit.
⋯⋯⋯⋯⋯⋯⋯⋯⋯⋯
本章记着小小的仪式：一小时阅读、慢慢回覆、一次有意的拒绝。每个都是伪装成习惯的策略。

Author / Deployment (click to expand)

Files included: This single-file site acts as index.html. You can split into /chapters/*.html and /assets/style.css later.

Quick Git commands (run in a terminal):

# create repo locally
mkdir the-sovereign-strategist && cd the-sovereign-strategist
# copy this file as index.html into this folder
git init
git add index.html
git commit -m "Initial site: index + chapter 1"
# create repo on GitHub named the-sovereign-strategist, then:
git remote add origin git@github.com:YOUR_USERNAME/the-sovereign-strategist.git
git branch -M main
git push -u origin main

Deploy on Netlify:

  1. Sign in to Netlify with GitHub and click "Add new site → Import from Git".
  2. Select this repository and deploy. Netlify will publish the site and provide a subdomain.

Or deploy with GitHub Pages by placing this file on the main branch and enabling Pages in repo Settings > Pages > Deploy from branch.

Tip: Keep your master copy of the story in a `content/` folder as Markdown, then convert to static HTML later with a small build step (Eleventy/Hugo) if you want more automation.
